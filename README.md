# cdmx_house_price

Se trata de una sencilla aplicación que permite visualizar datos rerlacionados con el precio de las vivicnedas en diferentes zonas de la CDMX.

Utilizamos un dataset obtenido de kaggle
https://www.kaggle.com/datasets/renecardoso/housing-prices-in-cdmx/data

La idea es explorar las herramientas y funciones de la libreria Streamlit y la generación de gráficos con la libreria Plotly.

Se ha hehco una limpieza y analisis minimo de los datos para que los resultados visuales sean coherentes.

Los datos mostrados pueden o no ser fieles a la realidad.
